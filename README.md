# SafeTrail-AI

SafeTrail-AI is a smart tourist safety monitoring and incident response platform built for travelers in high-risk and remote regions.

## Features

- Digital tourist ID
- KYC-based registration
- Trip itinerary management
- Safety score monitoring
- Geo-fencing alerts
- SOS / emergency support
- Tourist dashboard
- Admin / authority dashboard
- Map-based safety view

## Tech Stack

- React
- TypeScript
- Vite

## Run Locally

```bash
npm install
npm run dev
